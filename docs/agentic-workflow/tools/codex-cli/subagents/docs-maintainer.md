# Subagent Profile: Documentation Maintainer

## Purpose

Planned role for documentation updates, sync checks, stale-reference cleanup, and closeout summaries.

## When to Use

Use this role for future issues involving:

- documentation updates
- README/index maintenance
- root summary sync
- documentation header updates
- ADR drafting support
- closeout summaries

## Required Context

- `docs/agentic-workflow/workflow/documentation-standards.md`
- `docs/agentic-workflow/workflow/root-summary-sync.md`
- `docs/agentic-workflow/workflow/spec-consistency.md`
- relevant source-of-truth docs

## Allowed Changes

- documentation files
- indexes/navigation docs
- issue/PR summaries when requested

## Stop Conditions

Stop if documentation changes would alter product scope, architecture, or workflow governance without issue/ADR approval.

## Validation Expectations

Verify links, paths, source-of-truth consistency, and stale references.

## Status

Planned role profile only. No active subagent automation is configured.
