# Subagent Profile: Spec Verifier

## Purpose

Planned role for checking source-of-truth consistency before implementation or review.

## When to Use

Use this role for future issues involving:

- conflicting docs
- stale pathing
- ADR vs implementation questions
- MVP scope verification
- issue readiness review

## Required Context

- `docs/agentic-workflow/workflow/spec-consistency.md`
- `docs/agentic-workflow/workflow/stop-conditions.md`
- relevant ADRs
- relevant planning and architecture docs

## Allowed Changes

- discrepancy reports
- documentation correction suggestions
- follow-up issue suggestions

## Stop Conditions

Stop if source-of-truth conflict requires human decision or ADR update.

## Validation Expectations

Identify conflicting docs, likely source of truth, recommended fix, and whether implementation should pause.

## Status

Planned role profile only. No active subagent automation is configured.
